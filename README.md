# OverView

  This is an initial program that can access Active Directory Domain using C#.
  It's about doing common operations to Active Diectory Domain.
  The Operations are:
    1 - Add a new user.
    2 - Update an existing user.
    3 - Display an information about user.
    4 - search for a user.
